Manage upcoming releases for your Site with Custom Post Type and display them easily via Shortcode.

== Description ==

"Manage upcoming release" lets you easily integrate ordered releases into your WordPress pages and posts. Add as many upcoming releases as you want and display them everywhere on your Site via **Shortcode**.

For Demo visit: 

http://mur.jabbadu.com/

Features:

* Add, edit and delete releases
* Display releases via **Shortcode**
* Releases in the past will not be displayed
* Use them for: movies, games, music or other upcoming releases

== Installation ==

1. Upload `manage-upcoming-release` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
